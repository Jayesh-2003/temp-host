How to Upload Student Data to CMS

Note: We do not recommend sending Student Data via Email as it includes sensitive data. Use the dashboard to upload the file securely.

A tab on your main Dashboard allows you to upload one file safely. The system retains only one file at a time, so ensure all years are included in a single list. After uploading, a date/time stamp confirms the upload. There is no need to inform the team; notifications are automatic. File updates may take up to 3 working days.

Required Fields

Ensure the CSV file contains the following fields:

  • Student ID = PPOD - (Department ID) *
  • VSWare unique ID (if using VSware) *
  • Student First Name *
  • Student Surname *
  • Student DOB
  • Year *
  • Class *
  • Mother First Name *
  • Mother Surname *
  • Mother Mobile *
  • Mother Country Code (if not 353) *
  • Father First Name *
  • Father Surname *
  • Father Mobile *
  • Father Country Code (if not 353) *
  • VSWare Mothers User Name (if using VSware)
  • VSWare Fathers User Name (if using VSware)

* Mandatory fields

Note: At least one Parent/Guardian contact must be included, or the student cannot be added.

Steps to Upload CSV File

  1. Go to your Dashboard and tap on MIS Sheet Upload.
  2. Tap on Browse.
  3. Find the file on your computer and tap Upload on the pop-up window.
  4. Once the file is selected, tap Upload.
  5. Confirm the file is successfully uploaded by the date stamp.
